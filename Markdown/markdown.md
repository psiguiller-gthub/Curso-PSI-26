Escribir en Markdown es como redactar texto plano, pero usando símbolos sencillos para darle estructura y estilo. Al guardar tu archivo con la extensión .md, cualquier lector compatible (como GitHub, Notion o Obsidian) lo mostrará con formato. 
Aquí tienes los fundamentos para empezar a escribir hoy mismo:
# tamaño 1
## tamaño 2
### tamaño 3

1. Títulos y Estructura
Usa el símbolo de almohadilla (#) seguido de un espacio. Cuantas más almohadillas pongas, más pequeño será el subtítulo. 
# Título Principal (H1)
## Subtítulo (H2)
### Sección (H3)
2. Formato de Texto
Aplica estilos rápidamente rodeando tus palabras con asteriscos o guiones bajos: 
Negrita: **texto** o __texto__
Cursiva: *texto* o _texto_
Tachado: ~~texto~~
Negrita y cursiva: ***texto*** 
3. Listas y Tareas
Para organizar tus ideas, puedes usar números o viñetas: 
Lista desordenada: Usa -, * o + antes de cada elemento.
Lista ordenada: Escribe el número seguido de un punto (ej. 1., 2.).
Lista de tareas: Usa - [ ] para tareas pendientes y - [x] para marcarlas como hechas. 
4. Enlaces e Imágenes
La sintaxis es muy similar; solo cambia el signo de exclamación al principio para las fotos: 
Enlace: [Texto del enlace](https://url.com)
Imagen: ![Descripción de la imagen](ruta/o/url/de/imagen.jpg)
5. Código y Citas
Si necesitas resaltar comandos o frases de otros: 
Código en línea: Envuelve el texto con una comilla invertida (`).
Bloque de código: Usa tres comillas invertidas (```) al principio y al final del bloque.
Citas: Empieza la línea con el símbolo mayor que (>). 
6. Tablas
Se construyen usando barras verticales (|) para separar columnas y guiones (---) para la fila del encabezado.

Consejos Pro
Salto de línea: Pulsa Enter dos veces para crear un nuevo párrafo.
Línea horizontal: Escribe tres guiones seguidos --- en una línea vacía.
Previsualización: Si usas Visual Studio Code, presiona Ctrl + Shift + V para ver cómo queda tu documento en tiempo real.
